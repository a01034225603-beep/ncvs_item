====================================================
 NCVS UDP 헬스체크 모니터 — Windows 포터블 패키지
====================================================

■ 시스템 요구사항
  - Windows 10 / 11 (64비트)
  - 추가 설치 불필요 (Python, Node.js 내장)

■ 최초 실행 순서 (1회만)
  1. install.bat  더블클릭
     → Python 패키지 설치
     → DB 초기화
     → 관리자 계정 생성 (admin / admin)

■ 일반 실행 순서
  1. start.bat  더블클릭
     → 서버 시작 후 브라우저 자동 오픈
  2. http://localhost:3000 에서 로그인
     - 계정: admin / 비밀번호: admin

■ 서버 종료
  - stop.bat  더블클릭

■ 데이터 초기화
  - data\ncvs.db 파일 삭제
  - install.bat 재실행

■ 주의사항
  - 포트 3000, 8000 이 사용 가능해야 합니다
  - 방화벽에서 해당 포트를 막으면 접속 불가
  - 이 패키지의 데이터는 운영 서버와 완전히 독립됩니다

■ 문의
  개발팀에 문의하세요.
